# Recycling Used or Old Things ♻️

An AI-based recycling assistance application designed to simplify waste management and promote sustainable living. The app helps users identify recyclable items, locate nearby recycling centers, and receive proper disposal guidance through an easy-to-use mobile interface.

## Features
- AI-powered recyclable item identification
- Image upload and analysis
- Recycling center locator using GPS
- Disposal guidance for non-recyclable items
- User authentication system
- Recycling awareness and educational support


## Technologies Used
- Adalo
- AI-based image recognition concepts
- GPS integration
- Mobile application design
- Design Thinking methodology

## Modules
1. Authentication Module
2. Image Upload & Analysis Module
3. Result & Disposal Guidance Module

## Objective
The project aims to make recycling more accessible, efficient, and engaging while encouraging environmentally responsible behavior and sustainable waste management practices.
